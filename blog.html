  • 1. Variable delcared with var can be accessed from anywhere in the document, be it within a block scope or global scope. Var can be reclared and reassigned.

Arrow function and regular function are different both in sytanx and their use. Arrow function can use implicit return and parenthesis can be omitted if there is only one parameter. Arrow functions can also be used as methods.

map(), forEach(), filter() and find() are array methods and they can be used in a variety of context.
  • map() can return a new array transforming the given array.
  • forEach() is used for looping through the elements of an array like other loops but with a easy to follow syntax.
  • filter() is used for selecting a subset of a given array.
  • find() is used for searching a single element of an array
